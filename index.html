How to play

  • Find a buddy and choose which one is Bud #1 and which one is Bud #2
  • Go to https://github.com/njall/rock-paper-scissors
  • Bud #1: Import project and name it rock-paper-scissors
  • Bud #1: Add Bud #2 as a collaborator
  • Both open up your terminal and run this command to download the code: 
    git clone git@github.com:bud-1-username/rock-paper-scissors.git
  • Open your weapon of choice: _data/rock.yml, _data/paper.yml or _data/scissors.yml. Change false to true from your chosen file and add your name.
  • Run this command: 
    git add _data/file.yml
    where file is your choice (e.g. rock). This 'stages' the file which tells git 'Hey, I think this file is ready to be saved' read more)
  • Run this command: 
    git commit -m "Added my weapon of choice"
    which tells git 'Yep, go ahead and save this on my machine, and here's a handy reminder of what I've done'
  • Run this command: 
    git push
    which tells git 'Now push those changes onto github for the whole world to see'
  • An issue occured for the slow-poke! You've both edited the same repository at the same time. You can add one person's set of changes, but the second one requires a bit of action!
  • Slow-poke, run this command: 
    git pull
    you've pulled Speedy Gonzalez' changes into your repository
  • Now you need to accept these changes. Try committing them again like you did when you chose your weapon the first time. When you're ready, push your changes to the repository!
  • Not good? Might be a merge error!
  • Finally, go to bud-1-username.github.io/rock-paper-scissors to see who won!
